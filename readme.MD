## KotlinRectBridgeView.

### Custom View in android created using Kotlin.

### Usage

#### In app/build.gradle.
```
    compile project(":rectbridgeview")
```

#### In app/src/main/packagename/MainActivity.

```
    RectBridgeView.create(this)
```
